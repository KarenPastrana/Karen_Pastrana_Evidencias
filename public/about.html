Previous work

One of the projects I have done was a biped robot, initially it was to simulate an exoskeleton, but in the end it only remained as a biped because we wanted to simulate the gait cycle, which is much more complex than we can imagine. The name of the biped robot was Pinocho.​

Another project I did was a small car controlled by PWM and with the ability to receive commands via Bluetooth interface. The name of the car was Ramona.​

I also participated in the creation of a device called Bioarmband, which consisted of using a temperature sensor MLX90614, and the Max30102 sensor to create a device with pulse rate and estimates of SpO2 - blood oxygen percentage.​
