6. Electronics design

For this week the assignment was Write a program for a microcontroller development board to interact (with local input &/or output devices) and communicate (with remote wired or wireless devices).

I used the XIAO-RP2040 microcontroller using Python and C++, although I have also used the ATmega328P, and I will show an example of a project I have completed. The FabLab at Ibero Puebla provided us with some codes so that we could compare on our own and determine the programming language we can use.

You can find the group assignment page here.

Research

The XIAO-RP2040 microcontroller stands as a versatile and powerful option for embedded system development, offering a blend of performance and flexibility. Powered by the RP2040 microcontroller chip developed by Raspberry Pi, it inherits the robust capabilities of the RP2040 while maintaining a compact form factor suitable for various projects. One of the notable features of the XIAO-RP2040 is its compatibility with popular programming languages such as Python and C++. This broad language support facilitates rapid development and enables users with diverse programming backgrounds to harness the full potential of the microcontroller.

Python, renowned for its simplicity and readability, provides an accessible entry point for beginners and seasoned developers alike. With its rich ecosystem of libraries and frameworks, Python empowers users to efficiently implement a wide range of applications on the XIAO-RP2040, spanning from simple sensor interfacing to complex automation tasks.

On the other hand, C++ offers unparalleled performance and low-level control, making it an ideal choice for projects that demand optimized resource utilization and real-time responsiveness. With C++, developers can harness the full computational capabilities of the XIAO-RP2040, delving into advanced hardware manipulation and system-level programming.

The ATmega328P microcontroller is developed by Microchip Technology, featuring an 8-bit architecture and a wide range of integrated peripherals. Its compact design and low power consumption make it ideal for applications requiring standalone operation with limited resources, such as control systems, measuring devices, and portable gadgets. One of the distinctive features of the ATmega328P is its compatibility with the C programming language and the Arduino integrated development environment (IDE). This combination enables developers, from beginners to experts, to quickly prototype and develop complex projects intuitively and efficiently. Additionally, the active Arduino community provides a wide array of resources, libraries, and examples that streamline the development process and foster collaboration and knowledge sharing.

Code XIAO-RP2040

For the first part of programming the XIAO-RP2040, I used the blinking of two LEDs and it was programmed with Arduino.

Code ATmega328P

For the ATmega328P microcontroller, the project I carried out was to create a car that could be controlled via Bluetooth (forward, backward, right, left) with the purpose of navigating through the Ibero Puebla University campus. It is important to highlight that, in addition to the C code for mobility, the MIT App Inventor screen was also utilized to enable the use of a device such as a cellphone as the remote control.

The interesting aspect of this project is that through the MIT App Inventor app, a button on the screen could be translated to the Bluetooth module by using numbers. Upon verifying with the code, you can see that the PWM was configured to go in the direction indicated by the button.


Some of the elements I used were the programmer to upload the program I created in Visual Studio to the microcontroller, as well as a Bluetooth Module (H06), 3 rechargeable batteries of 1.5V each, 4 motors, and an H-bridge. The PCB was sent to be manufactured in China, and you can download the files to make your PCB from the downloadable section. :)
